# Logic Gate Simulator

## Overview
This project is a C++ based Digital Logic Gate Simulator that demonstrates the working of different logic gates using binary inputs.

## Gates Implemented
- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate

## Technologies Used
- C++
- Standard Library

## How to Run
1. Compile the program using a C++ compiler
2. Run the executable
3. Select the gate type
4. Enter binary inputs (0 or 1)
